# JustNoSinclair
Reddit bot for flagging links to Sinclair-owned websites.

## Requirements
* python3
* pipenv
* praw

## Setup
* git clone
* pipenv install
* create praw.ini
    [BotName]
    client_id=<client id>
    client_secret=<client secret>
    username=<bot reddit username>
    password=<bot reddit password>
    user_agent=<custom bot user agent>
